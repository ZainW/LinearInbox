# Homebrew Tap for LinearInbox

This is a Homebrew tap for [LinearInbox](https://github.com/ZainW/LinearInbox), a menu bar app for Linear issues.

## Installation

```bash
brew tap ZainW/linearinbox
brew install --cask linearinbox
```

## Updating

```bash
brew upgrade --cask linearinbox
```

## Note for Unsigned Apps
